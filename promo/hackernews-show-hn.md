## 正文

I'm a Taiwanese guy living on the Bōsō Peninsula in Japan. My neighbor built the world's first CLT (cross-laminated timber) treehouse — constructed around 5 living cedar trees, with a skylight bedroom, an outdoor rotenburo bath, and a forest sauna.

I also have 16 rescue cats.

I've been thinking about how open-source maintainers rarely get anything tangible for their work. Anthropic started giving Claude Max to top maintainers, which is great. I thought: I have a treehouse. I have cats. I have a car. What if I just... invited someone?

So I built a simple lottery:

- Your total GitHub stars (across all repos) = your number of tickets
- 10,000 stars → 10,000 tickets. 3 stars → 3 tickets.
- Stars are pulled automatically via the GitHub API
- Entry = open a GitHub Issue, check two boxes

The winner gets 3 days / 2 nights (June 7–9) for themselves + 3 friends. Private car from Tokyo, all meals, sauna, cats, the works.

The only requirement: you need to be a Claude Max subscriber (verified after winning). This is timed right before Code with Claude Tokyo on June 10.

Why "cat Treehouse"? `cat` is a Unix command. I have 16 actual cats. In a treehouse.

Details and photos: https://treeavel.washinmura.jp/cat-treehouse/

Enter: https://github.com/sstklen/cat-treehouse

Independent community event. Not affiliated with Anthropic.

## 如果被問常見問題（準備好在留言區回）

**"Why Claude Max only?"**
→ This treehouse stay is timed with Code with Claude Tokyo. It's a community thing for people in that ecosystem. I wanted a simple, verifiable filter.

**"Isn't this just marketing for your treehouse?"**
→ Fair question. I do run the treehouse as a guesthouse (treeavel.jp). But this giveaway is genuinely free — no strings, no content obligations. I just want to see what happens when you put devs in a forest with cats.

**"What's CLT?"**
→ Cross-laminated timber — engineered wood panels where layers are stacked at 90° angles and glued. It's as strong as concrete but way lighter. This treehouse is built around 5 living trees without harming them. Pretty cool engineering if you're into that sort of thing.

**"Can I just book normally?"**
→ Yes, about $1,100/person for a similar stay. But the lottery is free.
