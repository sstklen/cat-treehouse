## How the Draw Works

**More open source work = better odds.** Simple.

## How to Enter

1. Be a Claude Max subscriber
2. [Open an Issue →](https://github.com/sstklen/cat-treehouse/issues/new?template=enter.yml)
3. Check two boxes. Done.

That's it. We get your GitHub username from the Issue. Stars are checked automatically. No forms, no email, no hassle.

**Entries close May 31, 2026.**
